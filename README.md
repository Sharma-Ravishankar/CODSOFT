# CodSoft Python Programming Internship Projects

## 📌 Introduction
This repository contains the mini projects developed during my Python Programming Internship at CodSoft. These projects are designed to improve programming knowledge, logical thinking, and problem-solving skills using Python. Each project focuses on basic Python concepts and practical implementation.

---

## 🚀 Project Descriptions

### 1️⃣ Simple Calculator
The Simple Calculator is a basic Python application that performs arithmetic operations such as addition, subtraction, multiplication, and division. This project helps in understanding conditional statements, functions, and user input handling in Python.

### 2️⃣ Password Generator
The Password Generator is a Python program that creates strong and secure random passwords. Users can specify the desired password length, and the program generates passwords using letters, numbers, and special characters. This project demonstrates the use of Python modules like `random` and `string`.

### 3️⃣ To-Do List Application
The To-Do List Application is a simple task management program that allows users to add, view, and remove tasks. This project helps in understanding lists, loops, and function implementation in Python while creating a useful real-life application.
